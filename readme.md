# Estructura del Proyecto

```bash
src/
├── app/
│   ├── core/         # Servicios y clases que tienen que ver con la lógica de la app
│   ├── shared/       # Componentes reutilizables
│   ├── auth/         # Todo lo relacionado con la autenticación
│   ├── food/         # Lógica de negocio relacionada con los alimentos
│   ├── home/         # Módulo principal de la app
│   ├── guards/       # Guards para proteger rutas
│   └── state/        # Gestión del estado global de la aplicación
├── assets/           # Imágenes y otros recursos estáticos
├── environments/     # Configuraciones de ambiente (dev, prod)
└── styles/           # Archivos SCSS globales
```
